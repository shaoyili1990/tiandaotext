# tiandaotext 项目规划

## 项目目标
基于用户天道/人道系统的AI写作辅助工具，完全独立于SillyTavern运行。

## 核心功能需求

### 1. 天道系统（人物意识模拟）
- Y值系统：主观意识凝练强度(0-100)
- MBTI系统：16型人格骨架
- 三大机制：触发/补偿/回弹
- 记忆系统：5型记忆+PTSD
- 动机系统：7本能×4层级
- 心理学引擎：七情计算

### 2. 人道系统（人物关系与评判）
- 权重关系网：主角85-95/反派80-90/主要60-80/次要40-60/NPC 0-30
- 老天气变量：主观行为评判
- 蝴蝶效应：空间距离传导
- 因果网：事件关联影响

### 3. 功能系统
- 拆书：卷→章→角色快照+事件卡
- 续写：基于天道/人道约束生成
- 演化：Y值/记忆/权重/状态变化
